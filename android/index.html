Part1: Java 环境配置

通常通过在 Android SDK（Android软件开发包）中使用 Java 作为编程语言来开发 Android 应用程序。

1. 安装Java开发工具箱（JDK）

下载地址：https://www.oracle.com/java/index.html

[Java for Developers] --> [Java SE]

Java 运行时环境（JRE）包含虚拟机但不包含编译器

例如：Java SE 8u73——Java 标准版 8版第73次更新（内部版本号1.8.0_73）

修改安装路径：D:\Java\jdk1.8.0_*

安装路径最好不要含有空格（例如：D:\Program Files\Java\jdk）

2. 设置执行路径（添加环境变量）

执行路径是指操作系统搜索本地可执行文件的目录列表。

  • 我的电脑（右键）--> 属性
  • 高级系统设置 --> 高级 --> 环境变量...
  • 添加 Path: F:\Java\jdk1.8.0_73\bin

3. 编译和运行简单的Java程序

Tip：Shift+右键，可选择“在此处打开命令窗口”（cmd）
  • 写一个简单的java程序：任何文本编辑器都可以，文件命名为***.java
  • 打开命令行窗口（cmd）
  • 编译java源程序，会生成可执行的***.class文件：javac Welcome.java
  • 运行编译好的可执行文件：java Welcome

Java class文件是可以运行在任何支持Java虚拟机的硬件平台和操作系统上的二进制文件

4. Java 学习资料

书籍：《head first java》、《core java》（《Java核心技术》）

官方API文档：http://docs.oracle.com/javase/8/docs/api/
